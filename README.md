# LISA-segm

## Installation
```
pip install -r requirements.txt
pip install flash-attn --no-build-isolation
```

## Dataset Structure
In the root directory of this repo, create the following directory structure
```
├── dataset
│   ├── refer_seg
│   │   ├── images
│   │   |   └── mscoco
│   │   |       └── images
│   │   |           └── train2014
│   │   ├── refcoco
```
